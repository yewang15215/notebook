# PHP 学习笔记


* 遍历对象
    - 例如用 foreach 语句，默认情况下，所有**可见属性**都将被用于遍历。
* array_merge 和 array_push
    ```
    <?php
    $a = [[1, 2], [3, 4]];
    $b = [5, 6];
    $test = array_push($a, $b);
    ```
    则：
    ```
    $a === [[1, 2], [3, 4], [5, 6]];
    $b === [5, 6];
    $test === 3;
    ```

    ----
    ```
    $a = [[1, 2], [3, 4]];
    $b = [5, 6];
    $test = array_merge($a, $b);
    ```
    则：
    ```
    $a === [[1, 2], [3, 4]];
    $b === [5, 6];
    $test === [[1, 2], [3, 4], 5, 6];
    ```
